# Thermos AI

Thermos AI is a high-performance, enterprise-grade Urban Heat Island (UHI) mapping, analysis, and climate mitigation platform.

## Architecture Highlights
- **Backend Services**: Built using FastAPI, containing modules for auth, satellite image parsing, ML forecasting, and micro-simulations.
- **Frontend Application**: Developed with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Leaflet.js.
- **ML Engine**: Leverages PyTorch (U-Net) for spatial segmentation, and XGBoost/Prophet for forecasting future warming trends.
- **Databases**: PostgreSQL + PostGIS, MongoDB, and Redis.
